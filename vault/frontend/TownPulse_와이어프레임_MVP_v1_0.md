## 0. 스펙 정합 요약

| 문서 | 와이어프레임 반영 |
|---|---|
| ERD v5 | `SNAP_TRANSPORT`: `nearest_stop_distance_m`, `bus_stops_within_1km`, `avg_bus_interval_min` |
| 백엔드 v8.3 | 화면 API = **오케스트레이터** (`dashboard/*`, `village-detail`, `report-data`) |
| 프론트 v2 | URL `village_code`(10자리) · 내부 API `village_id`(UUID) |

**교통 공백 (TVI·대시보드 `transport_gap_count`):**  
`bus_route_count != null` 이고 (`bus_stops_within_1km === 0` 또는 `nearest_stop_distance_m > 1000`) → `bus_interval_score = 0`. `bus_route_count == null`(증평군) → 절충 점수·「교통 데이터 제한적」 배지 ★ v9.4

**데이터 수집 (백그라운드, 화면 비표시):** vworld geocode → 버스 노선 2단계 + 정류소 API(15098534)

### 1-1. MVP 정의 기준

대회 제출 기준 · 라이브 데모 시연 가능 상태를 목표로 한다.  
"실제 작동하는 최소 기능 세트"만 포함하며, 나머지는 명시적으로 제외한다.

```
핵심 데모 시나리오 (단양군 영춘면 예시)
─────────────────────────────────────────
1. 충북 히트맵 진입 → 단양군 영춘면 클릭
2. 진단: TVI 12점 · 빈집 34% · 배차 180분 · **정류장 1.2km** · 고령화 61%
3. AI 처방 3가지 자동 생성 (Gemini) + 예산 범위 표시
4. "PDF 리포트 출력" → 담당자 보고서 즉시 활용
```

## 7. API 연동 정의

> ★ v2: prefix `/api/townpulse/`. 상세 타입은 `TownPulse_프론트엔드_개발정의서_v2_3.md` §13.

### 7-1. 백엔드 연동 목록

| 화면 | API 엔드포인트 | 메서드 | 용도 |
|---|---|---|---|
| D-01 | `/api/townpulse/users/login` | POST | JWT 발급 |
| D-02 | `/api/townpulse/dashboard/summary` | GET | 지표 카드, TOP5, `transport_gap_count` |
| D-03 | `/api/townpulse/dashboard/map/villages` | GET | 히트맵 전체 목록 |
| D-03 | `/api/townpulse/dashboard/map/villages/{village_code}` | GET | 마을 요약 (정류장 거리) |
| D-04 | `/api/townpulse/village-detail/{village_code}` | GET | 지표 5종 + TVI + 처방 미리보기 |
| D-05 | `/api/townpulse/prescription-results/by-village/{village_id}` | GET | 처방 3개 |
| D-05 | `/api/townpulse/prescription-results` | POST | 처방 생성 |
| D-05 | `/api/townpulse/prescription-results/{id}/stream?token=` | GET | Gemini SSE |
| D-06 | `/api/townpulse/report-data/{village_code}` | POST | 리포트 데이터 |

### 7-2. 식별자 흐름

```
URL village_code (10자리)
  → village-detail → village_id (UUID)
  → prescription-results/by-village/{village_id}
```

### 7-3. 인증 헤더 규칙

```javascript
// 모든 API 요청에 포함
headers: {
  'Authorization': `Bearer ${localStorage.getItem('access_token')}`,
  'Content-Type': 'application/json'
}
```

### 7-4. CORS 도메인

- 프로덕션: `https://townpulse.site`, `https://www.townpulse.site`
- 개발: `http://localhost:3000`
